# MAGI_System
Multi-agent decision system inspired by EVA's MAGI — Balthasar (Claude) · Melchior (Gemini) · Caspar (GPT)

## 运作机制

三个单元完全平等，并行独立分析，互不知道对方的结果。采用一票否决制：三票全部通过才承認，任一拒绝即否决。

```
MELCHIOR·1 ──→ 通过/拒绝 ──┐
BALTHASAR·2 ──→ 通过/拒绝 ──┼──→ 一票否决制 ──→ 承認/否决
CASPER·3 ──→ 通过/拒绝 ────┘
（三票全通过才承認，任一拒绝即否决）
```
